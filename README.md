# Amazon Clone (Full Stack)

Full Stack Amazon Clone built with React, Spring Boot and MySQL.

## Tech Stack

Frontend
- React
- React Router

Backend
- Spring Boot
- REST API

Database
- MySQL
- Hibernate / JPA

## Features

- Product listing
- Add to cart
- Checkout page
- Login / Signup
- Search products
- Category filter

## Run Project

Backend

mvn spring-boot:run

Frontend

npm install
npm run dev

Open

http://localhost:5173
